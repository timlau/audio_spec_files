## RPM spec files and Makefile's to build audio applications & plugins for Fedora 42 & Later

Build packages are available in the [timlau/audio](https://copr.fedorainfracloud.org/coprs/timlau/audio/) Fedora Copr Repository

This repository contains the RPM spec files and Makefile's to build audio applications & plugins for Fedora 42 & Later.

Check this [webpage](https://timlau.github.io/posts/fedora-audio-timlau-audio/) for more information about the audio applications & plugins that are available in this repository.
